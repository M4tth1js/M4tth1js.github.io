Het geheugen

RAM

RAM staat voor Random Access Memory en is eigenlijk de opslag voor heel het werkgeheugen achter de computer. Het verschil tussen RAM en bijvoorbeeld een SSD-kaart waar je ook dingen in opslaat, is dat je in de SSD-Kaart dingen opslaat zoals afbeeldingen en teksten; in de RAM sla je juist handelingen op die de processor kan gebruiken om bepaalde handelingen te laten uitvoeren die op dat moment nodig zijn. Een nadeel van RAM is echter dat als 1 van de handelingen wordt uitgevoerd terwijl de computer zichzelf uitschakelt een deel van de RAM voor altijd verloren gaat. Dit is ook de reden dat je je computer niet mag afsluiten als hij aan het updaten is, anders gaat alle RAM verloren. De RAM capaciteit is bij een computer ook heel erg belangrijk, omdat de capaciteit bepaalt hoeveel handelingen je tegelijkertijd kan laten uitvoeren. Als je computer veel RAM heeft kan de computer veel verschillende handelingen of 1 heel grote ingewikkelde handeling beter uitvoeren zonder daarbij kwaliteit te verliezen.

ROM en het permanente geheugen

ROM staat voor Read Only Memory en lijkt eigenlijk niet op RAM terwijl de namen je misschien anders doen denken. ROM is zoals de naam al zegt alleen bedoeld om te lezen en niet om bepaalde handelingen uit te voeren zoals bij het RAM. ROM is ook in tegenstelling tot RAM permanent en kan ,zelfs als je dat graag zou willen, niet van je computer verwijderd worden. Het wordt namelijk gebruikt om bepaalde processen zoals de computer opstarten te gebruiken. Als je dit dus zou verwijderen dan zou je nooit meer je computer kunnen opstarten. Ram is hierdoor ook niet wijzigbaar en er kan ook niet overheen geschreven worden.

Er bestaan ook andere vormen van permanent geheugen zoals een SSD-kaart of een externe harde schijf. Op deze apparaten kan je geheugen opslaan om later weer te gebruiken of te bekijken. Deze gegevens kunnen wel verwijderd worden, maar dat moet je zelf doen en kan dus ook gebruikt worden om langdurig dingen om te slaan.
