Het CPU

Processor

De processor of de CPU (Central Processing Unit) is het belangrijkste deel van een computer dat verantwoordelijk is voor het uitvoeren van instructies en het verwerken van gegevens. Ook dit deel wordt samen met vele andere delen van de computer gezien als een deel van het brein van de computer of het belangrijkste gedeelte van de computer. Hierin worden alle gegevens, programma’s en instructies uitgevoerd, zodat de computer goed runt. De gegevens komen het CPU binnen door verschillende software die op dat moment berekeningen nodig hebben en doorsturen naar het CPU. Moderne CPU’s maken gebruik van kernen. Elke kern kan worden gezien als een werkunit die berekeningen kan maken en gegevens kan verwerken. Hoe meer kernen je hebt (dual-core of quad-core bijvoorbeeld), hoe meer gegevens je kan verwerken waardoor je meer en sneller je computer kan gebruiken.

Kloksnelheid en verwerkingssnelheid

De kloksnelheid van een processor verwijst naar het aantal bewerkingen dat de CPU per seconde kan uitvoeren. Het wordt uitgedrukt in hertz (Hz), waarbij 1 Hertz staat voor één bewerking per seconde. Processoren worden tegenwoordig gemeten in gigahertz (GHz), waarbij 1 GHz gelijk staat aan 1 miljard cycli per seconde. Een processor met een kloksnelheid van 3 GHz kan dus theoretisch 3 miljard bewerkingen per seconde uitvoeren. De kloksnelheid wordt bepaald door een interne klok in de processor, die de uitvoering van taken synchroniseert. Deze klok tikt met een vaste snelheid, en bij elke tik kan de processor een bepaalde hoeveelheid werk doen, zoals het ophalen van instructies, het uitvoeren van berekeningen of het verplaatsen van gegevens.

De verwerkingssnelheid van een processor is hoe snel deze gegevens en instructies kan verwerken. Het hangt niet alleen af van de kloksnelheid (het aantal bewerkingen per seconde), maar ook van factoren zoals het aantal kernen, de efficiëntie van de processorarchitectuur, en het cache-geheugen. Samen bepalen deze elementen hoe snel een processor taken kan uitvoeren.
